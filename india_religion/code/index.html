Religious diversity map of India

Insights into population and diversity of various religions across all the districts of India.

There are a total of 240,150 dots. Each dot represents 5,000 Indians. Zoom in to explore the diversity and the religions present in different regions of the country. Click on Religion to view the district wise population of different religions. Click on Diversity to view the district wise religious diversity index across the country.

All the code for this project can be found on Github.

The data for population of various religions for each district in India comes from 2011 Census. The dots are randomly placed within the district's boundaries. As a result, the position is accurate upto only district level and not taluk or any sub-district levels. Also because of random placement, some of the dots may get placed over nearby water bodies.

The geojson data for district boundaries in India was obtained from here. The diversity index for districts were computed using Simpson index which is a statistical measure used to determine the diversity in an area given the population distribution of different groups in that area.

This work wouldn't be possible without the help of various data analysis and visualisation tools such as jupyter, pandas, mapbox. The template for this html page was adapted from Start Bootstrap.
